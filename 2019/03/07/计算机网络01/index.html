计算机网络—第一章 计算机和因特网

1.1 什么是因特网

1.1.1 具体构成
  • 端系统（主机）：因特网由数以亿计的计算设备互联而成
  • 通信链路： 1.信道类型：光纤，同轴电缆，无线电，卫星
    2.传输速率：带宽
  • 分组交换：路由器 链路层交换机
  • 协议：用于控制因特网中的信息接受和发送 （TCP HTTP UDP….)
  • 因特网标准：RFC、IETF
1.1.2 服务描述
  • 因特网是为应用程序提供网络服务的基础设施
  • 为网络应用提供应用程序接口
  • 应用程序仅运行在终端上，不运行在网络核心的分组交换机中
    1.1.3 协议
    协议是通信计算机双方必须共同遵从的一组约定。如怎么样建立连接、怎么样互相识别等。只有遵守这个约定，计算机之间才能相互通信交流。它的三要素是：语法、语义、时序。
    [核心][df1]：报文的传输，接收以及所采取的动作

1.2 网络边缘

  • 网络边缘（资源子网）：外围部件，主机（客户，服务器），网络应用
  • 网络核心（通信子网）：路由器，通信链路，网络的网络
1.2.1 接入口
  • 接入网：将端系统连接到其边缘路由器的物理链路
  • 边缘路由器：端系统到任何其他远程端系统的路径上的第一台路由器
1.2.1.1 家庭接入

DSL,电缆，FTTH，拨号和卫星

  • 数字用户电路（DSL）：家庭常见的是ADSL（不对称数字用户线） 其特点是：1.采用双绞线 2.下载速度快 3.采用频分复用技术（实现调频频段的分离，同时打电话上网）
  • 光纤到户
  • 电缆因特网接入，又称混合光纤同轴（利用有线电视线路，一个重要特点是共享广播媒体）
  • 卫星链路接入
1.2.1.2 企业（学校）接入

局域网：先将多个端系统连接成局域网，局域网再与边缘路由器连接

  • 以太网
以太网技术
共享以太网 端系统共享以太网的传输速率
交换以太网 多个用户可同时使用全部带宽通信
  • 无线局域网
    无线用户通过基站连接 基站与有线的因特网连接，为无线用户提供服务 通过基站
    -广域无线接入
